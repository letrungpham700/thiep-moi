# Thư mục ảnh cưới

Đặt các ảnh theo tên sau vào thư mục này:

| File | Mô tả |
|------|-------|
| `hero-couple.jpg`   | Ảnh cặp đôi đứng (hero section, arch lớn) |
| `hero-romantic.jpg` | Ảnh lãng mạn (hero section, arch nhỏ) |
| `groom.jpg`         | Ảnh chú rể đơn |
| `bride.jpg`         | Ảnh cô dâu đơn |
| `love-story.jpg`    | Ảnh chuyện tình (arch lớn) |
| `album-1.jpg` → `album-6.jpg` | 6 ảnh gallery |
| `milestone-1.jpg`   | Ảnh lần đầu gặp nhau |
| `milestone-2.jpg`   | Ảnh chuyến đi đầu tiên |
| `milestone-3.jpg`   | Ảnh hẹn ước trọn đời |
| `icon-ring.png`     | Icon nhẫn cưới (events) |
| `icon-car.png`      | Icon xe hoa (events) |
| `icon-cake.png`     | Icon bánh cưới (events) |

> Kích thước đề xuất: ảnh portrait 800×1000px, ảnh landscape 1200×800px
